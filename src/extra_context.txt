# 자동차 정비 RAG 참고 데이터

## 차종별 고질병 정보

### 현대 LF쏘나타 (2014-2019)
- 타이밍 체인 텐셔너 이상으로 시동 시 소음 발생 가능
- 앞 서스펜션 로어암 부시 조기 마모 (10만km 전후)
- 예상 수리비:
  * 타이밍 체인 텐셔너 교체: 부품 20-30만원 + 공임 30-40만원
  * 로어암 부시 교체: 부품 8-12만원 + 공임 10-15만원

### 기아 K5 (2010-2015)
- 스로틀 바디 이상으로 공회전 불안정
- 연료펌프 조기 고장 (15만km 전후)
- 예상 수리비:
  * 스로틀 바디 교체: 부품 25-35만원 + 공임 10-15만원
  * 연료펌프 교체: 부품 30-40만원 + 공임 15-20만원

### 쉐보레 스파크 (2010-2015)
- 점화코일 고장으로 출력 저하 및 떨림
- 엔진 마운트 조기 마모로 진동 심화
- 예상 수리비:
  * 점화코일 교체: 부품 5-8만원 x 4개 + 공임 5-10만원
  * 엔진 마운트 교체: 부품 10-15만원 + 공임 10-15만원

### 쉐보레 크루즈 (2011-2016)
- 터보차저 오일 누유
- 냉각수 온도 센서 고장
- 예상 수리비:
  * 터보차저 교체: 부품 80-120만원 + 공임 40-60만원
  * 냉각수 온도 센서: 부품 3-5만원 + 공임 5-10만원

## 정비 주기 기본 정보

### 엔진오일 교체
- 가솔린: 10,000km 또는 6개월마다
- 디젤: 15,000km 또는 1년마다
- 예상 비용: 5-10만원 (합성유 기준)

### 브레이크 패드 교체
- 교체 주기: 30,000-50,000km
- 전륜: 8-15만원 (부품 + 공임)
- 후륜: 8-12만원 (부품 + 공임)

### 타이어 교체

- 국산 타이어: 개당 10-20만원
- 수입 타이어: 개당 20-40만원

### 배터리 교체
- 교체 주기: 3-5년
- 국산 배터리: 8-15만원
- 수입 배터리: 15-25만원

## 증상별 일반적인 원인과 수리비

### 시동 시 덜컹거림
- 가능한 원인: 엔진 마운트 마모, 타이밍 체인 텐셔너
- 예상 수리비: 20-80만원

### 브레이크 밟을 때 떨림
- 가능한 원인: 브레이크 디스크 변형, 패드 불량
- 예상 수리비: 15-40만원

### 방지턱 넘을 때 소음
- 가능한 원인: 쇼바 마모, 로어암 부시 손상
- 예상 수리비: 15-60만원

### 공회전 시 떨림
- 가능한 원인: 점화코일, 엔진 마운트, 스로틀 바디
- 예상 수리비: 10-50만원

### 냉각수 온도 상승
- 가능한 원인: 워터펌프, 서모스탯, 라디에이터
- 예상 수리비: 20-100만원
- ⚠️ 즉시 정비소 방문 필요

### 엔진 경고등 점등
- 가능한 원인: 산소센서, 촉매, 엔진 제어 시스템
- 예상 수리비: 10-200만원 (원인에 따라 변동 큼)
- ⚠️ 진단 후 조치 필요
